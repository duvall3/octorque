OVERVIEW

Software for importing OBDII data (via the Torque app) into Octave

USAGE

Short Version: - Clone this repository: git clone https://github.com/duvall3/octorque - Modify the OTD line in octorque.sh if needed - Either run the line PATH=$PATH:~/octorque/bin or add it to your .bashrc (modify directory if needed) - Navigate to the directory containing your Torque data - Run the command octorque.sh
